# installd8
Drupal 8 installer on Ubuntu/Debian (Commandline Tool)

# Installation
* $ cd /var/local
* $ git clone https://github.com/thom44/installd8.git
* $ cd installd8
* $ mv installd8 /usr/bin/installd8
* $ chmod u+x /usr/bin/installd8

# Usage
* $ sudo installd8

# Install with Composer 
* $ sudo installd8 -c
